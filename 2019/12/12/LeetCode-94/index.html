发表于 2019-12-12 | 分类于 LeetCode ， Normal

Question

Given a binary tree, return the inorder traversal of its nodes’ values.

Example:

Input: [1,null,2,3]
1
\
2
/
3

Output: [1,3,2]
Follow up: Recursive solution is trivial, could you do it iteratively?

Source：力扣（LeetCode）
Link ：https://leetcode-cn.com/problems/binary-tree-inorder-traversal

Solve

We can solve this question iteratively by using stack.

Firstly , we push all the left nodes into the stack , until cur is NULL.

Second , we pop the stack , so we can get an Not-NULL value , give it to cur.

Then , we add the cur.val into a vector. This is the mid in the mid (because left is null).

And then , let the cur = cur->right; , we can start traverse the right node.

That’s all.
