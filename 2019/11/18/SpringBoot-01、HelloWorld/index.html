一、Spring开发前准备

1、Maven环境搭建

Maven3.3版本以上都可以兼容

Spring Boot is compatible with Apache Maven 3.3 or above. If you do not already have Maven installed, you can follow the instructions at maven.apache.org.

2、Java版本

Java版本需求为jdk1.8及以上，现在已经兼容jdk13。

Spring Boot can be used with “classic” Java development tools or installed as a command line tool. Either way, you need Java SDK v1.8 or higher.

3、开发工具

  2. main函数中的启动语句SpringApplication.run(HelloWorldMainApplication.class,args);，这一句是调用SpringApplication的静态方法run，只需传入主类的.class以及可变参数即可。这一句的作用是使Spring应用启动我们所标注的主类。
