idea中jsp导入静态文件常见问题，看着一篇就够了

发表于 2019-05-29

错误信息

absolute paths not recommended in jsps.

这句话意思是在jsp文件中，不建议使用绝对路径来导入静态文件，如css，js等。

百度给出的结果

直接用$获得上下文路径

把

<script src="$/libs/jquery/jquery.min.js" ></script>

改为

<script src="${pageContext.request.contextPath}/libs/jquery/jquery.min.js" ></script>

即可

用JSP中的Java代码获取上下文路径

代码：

<%
String path = request.getContextPath();
String basePath = request.getScheme() + "://"
+ request.getServerName() + ":" + request.getServerPort()
+ path + "/";
%>

路径改为

<img src="<%=basePath%>images/head_success.jpg" height="80" width="650"/>

上述方法无法使用的情况

当web.xml中的webapp版本较低时，browser是无法解析 ${pageContext.request.contextPath} 的，浏览器指向的为$%7BpageContext.request.contextPath%7D，发生这种情况就是webapp版本过低，修改web.xml文件即可。

web.xml中不同版本的头部声明

  1. Servlet 3.1

Java EE 7 XML schema，命名空间是 http://xmlns.jcp.org/xml/ns/javaee/

<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns="http://xmlns.jcp.org/xml/ns/javaee" 
        xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
        xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/javaee
         http://xmlns.jcp.org/xml/ns/javaee/web-app_3_1.xsd"
        version="3.1">

</web-app>
  1. Servlet 3.0

Java EE 6 XML schema，命名空间是 http://java.sun.com/xml/ns/javaee

<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns="http://java.sun.com/xml/ns/javaee"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://java.sun.com/xml/ns/javaee
          http://java.sun.com/xml/ns/javaee/web-app_3_0.xsd"
          version="3.0">

</web-app>
  1. Servlet 2.5

Java EE 5 XML schema，命名空间是 http://java.sun.com/xml/ns/javaee

<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns="http://java.sun.com/xml/ns/javaee"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://java.sun.com/xml/ns/javaee
          http://java.sun.com/xml/ns/javaee/web-app_2_5.xsd"
          version="2.5">

</web-app>
  1. Servlet 2.4

Java EE 1.4 XML schema, 命名空间是 http://java.sun.com/xml/ns/j2ee

<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns="http://java.sun.com/xml/ns/j2ee"
          xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
          xsi:schemaLocation="http://java.sun.com/xml/ns/j2ee
          http://java.sun.com/xml/ns/j2ee/web-app_2_4.xsd"
          version="2.4">

</web-app>
  1. Servlet 2.3

Java EE 1.3 DTDs schema，这个版本的web.xml文件太老了，强烈建议升级

<!DOCTYPE web-app PUBLIC
 "-//Sun Microsystems, Inc.//DTD Web Application 2.3//EN"
 "http://java.sun.com/dtd/web-app_2_3.dtd" >

<web-app>
  <display-name>Servlet 2.3 Web Application</display-name>
</web-app>
