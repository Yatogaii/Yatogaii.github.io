LeetCode-349

发表于 2019-05-27 | 分类于 LeetCode

题目

给定两个数组，编写一个函数来计算它们的交集。

示例 1:

输入: nums1 = [1,2,2,1], nums2 = [2,2]
输出: [2]
示例 2:

输入: nums1 = [4,9,5], nums2 = [9,4,9,8,4]
输出: [9,4]
说明:

输出结果中的每个元素一定是唯一的。
我们可以不考虑输出结果的顺序。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/intersection-of-two-arrays

排序双指针法:

先讲一讲C++ sort函数的用法

对vector排序

sort(nums1.begin(),nums1.end());

对数组排序也是一样的用法，第一个参数为起始地址，第二个参数为最后一个元素的后一位的地址。

因为sort实际上是不去第二个参数所表明的地址的值的，就和vector.end()所指向的也不是最后一个值而是最后一个值的后一个位置一样。

删除C++中vector的重复元素

res.erase(unique(res.begin(), res.end()), res.end());

排序双指针法

执行用时 : 8 ms, 在Intersection of Two Arrays的C++提交中击败了99.49% 的用户

内存消耗 : 9.2 MB, 在Intersection of Two Arrays的C++提交中击败了65.37% 的用户

LeetCode中大佬最牛逼的解法

class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        vector<int> res;
        if(nums1.size() ==0 || nums2.size()==0)
            return res;

        sort(nums1.begin(),nums1.end());
        sort(nums2.begin(),nums2.end());

        int lastNum = -1;
        for(int i=0,j=0;i<nums1.size() && j<nums2.size();){
            if(nums1[i] == nums2[j]){
                if(lastNum !=nums1[i]){
                    res.push_back(nums1[i]);
                    lastNum = nums1[i];
                }                
                i++;
            }
            else if(nums1[i]<nums2[j]){
                i++;
            }
            else{
                j++;
            }            
        }
        return res;
    }
};
