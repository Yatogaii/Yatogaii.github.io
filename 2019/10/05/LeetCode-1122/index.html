LeetCode-1122

发表于 2019-10-05 | 分类于 LeetCode

题目

给你两个数组，arr1 和 arr2，

arr2 中的元素各不相同
arr2 中的每个元素都出现在 arr1 中

对 arr1 中的元素进行排序，使 arr1 中项的相对顺序和 arr2 中的相对顺序相同。未在 arr2 中出现过的元素需要按照升序放在 arr1 的末尾。

示例：

输入：arr1 = [2,3,1,3,2,4,6,7,9,2,19], arr2 = [2,1,4,3,9,6]
输出：[2,2,2,1,4,3,3,9,6,7,19]

提示：

arr1.length, arr2.length <= 1000
0 <= arr1[i], arr2[i] <= 1000
arr2 中的元素 arr2[i] 各不相同
arr2 中的每个元素 arr2[i] 都出现在 arr1 中

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/relative-sort-array

求解思路

经解答区的大神提醒，可以利用计数排序来对该题目进行排序，基本流程如下。

  1. 遍历arr1，记录数组中每一个数的出现的个数。（利用STL中的map实现）
  2. 通过遍历arr2，利用arr2的先后顺序结合map中arr1各元素的出现次数，对arr2中出现过的元素进行排序。
  3. 遍历map，对出现次数仍然>0的pair进行记录，排序。
  4. 利用insert操作，拼接出最后的答案。

代码

class Solution {
public:
    vector<int> relativeSortArray(vector<int>& arr1, vector<int>& arr2) {
        vector<int> res,tail ;
        map<int,int> arr1Map;
        //第一次遍历，记录arr1中各个元素出现的次数
        for(auto each : arr1){
            if(arr1Map.find(each) != arr1Map.end()){
                arr1Map[each] ++;
            } else {
                arr1Map[each] = 1;
            }
        }
        //根据arr2进行排序
        for(auto each : arr2){
            while(arr1Map[each] > 0){
                arr1Map[each] -= 1;
                res.push_back(each);
            }
        }
        //对最后的末尾进行排序
        for(auto pair : arr1Map){
            while(pair.second > 0){
                pair.second -= 1;
                tail.push_back(pair.first);
            }
        }
        sort(tail.begin(),tail.end());
        res.insert(res.end(),tail.begin(),tail.end());
        return res;

    }
};

结果

学到了什么

  • 计数排序方法：利用map记录各个元素出现的值，通过这个值来表示重复次数，之后进行包含重复项的排序。
  • 根据某一数组的顺序进行排序，直接遍历这个顺序数组，根据遍历的顺序直接push_back相应的值即可得到答案。
  • 对C++的STL中的map、pair和vector有了更深入的了解。
