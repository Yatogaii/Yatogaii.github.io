C++中各种变量前符号的作用

发表于 2019-09-06 | 分类于 C++

&

  • 学名：引用。

  • 引用必须被初始化。

    int ival = 1024;
int %refval = ival; //refVal指向ival(是ival的另一个名字) 
int & refVal2;      //报错，引用必须被初始化
int &ref = 10;      //错误，引用的值必须是一个对象。
  • 一旦初始化完成，引用将和他的初始化对象一直绑定在一起。无法令引用重新绑定到另一个对象上。
  • 引用本身不是对象，不能定义引用的引用。
  • 引用的值和对象本身的值是同步变化的。

作为参数时

函数的参数有引用时，可以直接通过函数内部操作来改变这个变量的值，函数调用本身不需要任何操作，只需要在函数本体上加上’&’。
例如:

void swap(int &x1,int &x2){
int tmp = x1^x2;
x1 = x1 ^ tmp;
x2 = x2 ^ tmp;
}
int x1=10,x2=5; //声明、调用，不需要任何操作，也不用取地址。
swap(x1,x2);    //即可满足要求

*

  • 学名：指针。
  • 指针存放某个对象的地址，获取地址需要用到取地址符，’&’。

  • 指针访问一个对象需要用解引用符’*’。

    int val = 10;
int *p = &val;
cout << *p ;
  • 因为指向同一个地址，指针和其本身的值也是同步改变的。
