Redis密码设置

发表于 2019-07-11

Redis的安装

在Ubuntu下安装redis很轻松

apt-get install update
apt-get install redis-server

之后输入命令

redis-server
redis-cli

即可确认是否安装成功

Redis密码的设置

进入redis命令行之后，输入 

config get requirepass

可以看到密码是空

config set requirepass 密码

即可设置密码

Redis的远程登录

仅设置密码还无法远程登录redis，因为redis一开始是只允许在本机登陆的，需要在redis的config里进行相应的设置。

  1. 注释掉Bind 127.0.0.1
  2. 将protected-mode 设为no
  3. 设置远程链接密码
