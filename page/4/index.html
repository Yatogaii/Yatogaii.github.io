LeetCode-905

发表于 2019-09-27 | 分类于 LeetCode

题目

给定一个非负整数数组 A，返回一个数组，在该数组中， A 的所有偶数元素之后跟着所有奇数元素。

你可以返回满足此条件的任何数组作为答案。

示例：

输入：[3,1,2,4]
输出：[2,4,3,1]
输出 [4,2,3,1]，[2,4,1,3] 和 [4,2,1,3] 也会被接受。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/sort-array-by-parity
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

求解

代码

class Solution {
public:
    vector<int> sortArrayByParity(vector<int>& A) {
        vector<int> ji;
        vector<int> ou;
        for(auto each : A){
            if(each % 2 == 0)
                ou.push_back(each);
            else
                ji.push_back(each);
        }
        ou.insert(ou.end(),ji.begin(),ji.end());
        return ou;
    }
};

学到了什么

可以利用如下的方法来合并两个Vector:

a.insert(a.end(),b.begin(),b.end());

位运算的妙用

发表于 2019-09-27

位运算

原理

计算机是利用二进制来存储数据的，使用位运算往往会有更高的效率。

常用操作符

符号 操作
<< 左移
>> 右移
~ 取反
\ 按位或
& 按位与
^ 按位异或

实际应用

利用异或实现两个数的交换

通常利用中间变量来实现交换两个数，如下

int tmp = a;
a = b;
b = a;

如果使用异或，也可以对整数进行交换。

int tmp = a ^ b;
a = tmp ^ a;
b = tmp ^ b;
//也可以不设置临时变量
a = a ^ b;
b = a ^ b;
a = a ^ b;

原理分析：

  1. 所有数与自己本身异或，其结果恒为0。
  2. 任何数和0异或，其结果恒为其本身。

陷阱：

当用异或法交换两个相同的数时，得到的最终结果是两个数都为0，所以需要进行是否相同的判断。

LeetCode-476

发表于 2019-09-27 | 分类于 LeetCode

题目

给定一个正整数，输出它的补数。补数是对该数的二进制表示取反。

注意:

给定的整数保证在32位带符号整数的范围内。
你可以假定二进制数不包含前导零位。

示例 1:

输入: 5
输出: 2
解释: 5的二进制表示为101（没有前导零位），其补数为010。所以你需要输出2。

示例 2:

输入: 1
输出: 0
解释: 1的二进制表示为1（没有前导零位），其补数为0。所以你需要输出0。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/number-complement
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

分析

5的二进制位为101

7的二进制位为111

5和7异或即为所求补数

所以只需要找到所给数字同样长度的数，使其减去或者异或即可达到答案。

实现

找到与所给数目同样长度的全为1的数字

int tmp = 1;
while(tmp<num){
    tmp <<= 1;
    tmp +=  1;
}

这样就能找到同样长度的全为1的数。

求补数

可以利用tmp与num相减，也可以利用tmp与num异或。

异或执行速度更快，建议使用异或。

代码

class Solution {
public:
    int findComplement(int num) {
        int tmp = 1;
        //让tmp和num同样长度
        while(tmp<num){
            tmp <<= 1;
            tmp +=  1;
        }
        return (tmp^num);
    }
};

学到的东西

  • 找到与所给数目同样宽度的数可以利用while循环，每次移位加一。
  • 全为1的数减去一个数，与全为1的数与这个数异或，结果一样，并且异或效率更高。

class ManHuaDui:

        self.fitstUrl = path

    #获取Html数据
    def getHtml(self):
        response = requests.get(self.fitstUrl)
        if response.status_code == 200:
            return response.text
        raise Exception

        pass

    #解析html获得密文和路径等数据
    '''
    @操作了chapterImages、Path、Name三个类变量
    '''
    def parseHtml(self):
        text = self.getHtml()
        #三个正则表达式
        reImages = 'chapterImages = "[^"]*"'
        rePath = 'chapterPath = "[^"]*"'
        reName = 'pageTitle = "[^"]*"'
        #进行匹配
        images = re.findall(reImages,text)[0]
        path = re.search(rePath,text).group(0)
        name = re.findall(reName,text)[0]
        #对匹配结果进行切片处理
        lockData = images[len('chapterImages = "'):-1]
        self.path = path[len('chapterPath = "'):-1]
        self.name = name[len('pageTitle = "'):-1]
        print(lockData)
        print(self.path)
        print(self.name)
        self.chapterImages = self.str2list(self.decryption(lockData))
        print(self.chapterImages)

    #对数据进行解密
    def decryption(self,data):
        # aes算法加密的密钥，加密模式为CBC，偏移量为Pks7
        key = '123456781234567G'
        iv = 'ABCDEF1G34123412'
        mode = AES.MODE_CBC
        #初始化解密器
        cryptos = AES.new(key.encode('utf8'), mode, iv.encode('utf8'))
        # 优先逆向解密base64成bytes
        data = data.encode('utf8')
        base64_decrypted = base64.decodebytes(data)
        # 执行解密密并转码返回str
        decrypted_text = str(cryptos.decrypt(base64_decrypted), encoding='utf-8').replace('\0', '')
        print(decrypted_text)
        return decrypted_text

    def str2list(self, str) -> list:
        res = str[1:-2].split(',')  #一定要是1:-2，因为原始字符串最后有一个空格，python的截取是不包括最后一个字符的
        for i in range(0,len(res)):
            res[i] = res[i].strip('"')
        res[-1] = res[-1].strip('"')
        return res

LeetCode-108

发表于 2019-09-17 | 分类于 LeetCode

题目

将一个按照升序排列的有序数组，转换为一棵高度平衡二叉搜索树。

本题中，一个高度平衡二叉树是指一个二叉树每个节点 的左右两个子树的高度差的绝对值不超过 1。

示例:

给定有序数组: [-10,-3,0,5,9],

一个可能的答案是：[0,-3,9,-10,null,5]，它可以表示下面这个高度平衡二叉搜索树：

 0
/ \

-3 9
/ /
-10 5

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/convert-sorted-array-to-binary-search-tree
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

求解

一个基本的区间分治类问题，以前听说过分治但是很少用。这个题的主要思想还是递归时传入左右边界，取其中点作为树的节点，之后分别对其左右进行递归。

代码

/**
* Definition for a binary tree node.
* struct TreeNode {
*     int val;
*     TreeNode *left;
*     TreeNode *right;
*     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
* };
*/
class Solution {
public:
    TreeNode* sortedArrayToBST(vector<int>& nums) {
        if(nums.size() == 0) return NULL;
        return sortedArrayToBST(nums,0,nums.size()-1);
    }
    TreeNode* sortedArrayToBST(vector<int>& nums,int l,int r){
        if(r<l)
            return  NULL;
        int mid = l + ((r-l)>>1);
        TreeNode* root = new TreeNode(nums[mid]);
        root->left = sortedArrayToBST(nums,l,mid-1);
        root->right = sortedArrayToBST(nums,mid+1,r);
        return root;
    }
};

LeetCode-944

发表于 2019-09-10 | 分类于 LeetCode

题目

给定由 N 个小写字母字符串组成的数组 A，其中每个字符串长度相等。

删除 操作的定义是：选出一组要删掉的列，删去 A 中对应列中的所有字符，形式上，第 n 列为 [A[0][n], A[1][n], …, A[A.length-1][n]]）。

比如，有 A = [“abcdef”, “uvwxyz”]，

要删掉的列为 {0, 2, 3}，删除后 A 为[“bef”, “vyz”]， A 的列分别为[“b”,”v”], [“e”,”y”], [“f”,”z”]。

你需要选出一组要删掉的列 D，对 A 执行删除操作，使 A 中剩余的每一列都是 非降序 排列的，然后请你返回 D.length 的最小可能值。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/delete-columns-to-make-sorted
著作权归领扣网络所有。商业转载请联系官方授权，非商业转载请注明出处。

解题

这题没什么好说的，逐行比较每一列是不是非降序排列就行了

贴代码：

auto __ =[]()
{
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr);
  return nullptr;
}();

class Solution {
public:
   int minDeletionSize(vector<string>& A) {
        int res = 0;
        int COL = A.size();
        int ROW = A[0].size();
        for(int i=0;i<ROW;i++){
            for(int j=0;j<COL-1;j++){
                if(A[j][i]>A[j+1][i]){
                    res ++;
                    break;
                }
            }
        }
        return res;
    }
};

C++中各种变量前符号的作用

发表于 2019-09-06 | 分类于 C++

&

  • 学名：引用。

  • 引用必须被初始化。

    int ival = 1024;
int %refval = ival; //refVal指向ival(是ival的另一个名字) 
int & refVal2;      //报错，引用必须被初始化
int &ref = 10;      //错误，引用的值必须是一个对象。
  • 一旦初始化完成，引用将和他的初始化对象一直绑定在一起。无法令引用重新绑定到另一个对象上。
  • 引用本身不是对象，不能定义引用的引用。
  • 引用的值和对象本身的值是同步变化的。

作为参数时

函数的参数有引用时，可以直接通过函数内部操作来改变这个变量的值，函数调用本身不需要任何操作，只需要在函数本体上加上’&’。
例如:

void swap(int &x1,int &x2){
int tmp = x1^x2;
x1 = x1 ^ tmp;
x2 = x2 ^ tmp;
}
int x1=10,x2=5; //声明、调用，不需要任何操作，也不用取地址。
swap(x1,x2);    //即可满足要求

*

  • 学名：指针。
  • 指针存放某个对象的地址，获取地址需要用到取地址符，’&’。

  • 指针访问一个对象需要用解引用符’*’。

    int val = 10;
int *p = &val;
cout << *p ;
  • 因为指向同一个地址，指针和其本身的值也是同步改变的。

Unity中必须掌握的概念

坐标

  • 世界坐标：整个场景的固定坐标，不会改变。（类似东南西北）
  • 本地坐标：物体本身的坐标，会随着物体的旋转而改变。（类似于生活中的前后左右）右X前Z上Y

    场景

    一组相关联的游戏对象的集合。

    游戏对象

  • 游戏运行时出现在场景中的游戏物体
  • 是一种容器，可以挂载组件。
  • 父、子物体
    子物体的坐标是相对于父物体的

    子物体集成父物体的移动，旋转和缩放属性，但子物体不影响父物体。

组件

  • 游戏对象的功能模块。
  • 每个组件都是一个类的实例。
  • 常见组件
    • Transform变换组件：决定物体位置，旋转、缩放比。
    • Mesh Filter网格过滤器：用于从资源中获取网格信息。 决定形状
    • Mesh Renderer网格渲染器：从网格过滤器中获得几何形状，然后根据变化组件进行渲染。
      可由里面的material属相来决定外观。
  • 可以在Inspector里的Add Compinent来为一个游戏对象添加组件。

Material

  • 材质
    • 可以直接在project面板出新建，拖到有Mesh Renderer组件的对象上，更改颜色之类的属性。
  • Shader着色器
  • Texture纹理：附加到物体表面的贴图。
    • 也可以直接拖动到物体上，原理上就是给到了物体的材质 -> Mesh Renderer组件上。

对应关系

Project->Scene->GameObject->Component

重点学习的是组件
