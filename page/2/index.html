本来是思修暑期调查的问卷，但是一个个找别人发太讨人厌了，就用python的随机数生成结果

生成结果代码

import random
double = [3,7,9]
mappings = {}
for times in range(1,51):   #50次伪证
    for No in range(1,11):   #生成10次随机数，对应十个题
        if No in double:    #如果是2选1题
            if random.randint(1,101) in range(1,51):     #选A
                key = '{d}A'.format(d=No)
                if key in mappings:
                    mappings[key] += 1
                else:
                    mappings[key] = 1
            else:                                   #选B
                key = '{d}B'.format(d=No)
                if key in mappings:
                    mappings[key] += 1
                else:
                    mappings[key] = 1
        else :              #那么就是三选一的题了
            randNum = random.randint(1,101)
            if randNum in range(1, 34):  # 选A
                ans = 'A'
            elif randNum in range(34, 67):
                ans = 'B'
            else:
                ans = 'C'
            key = '{d}{ans}'.format(d=No,ans=ans)
            if key in mappings:
                mappings[key] += 1
            else:
                mappings[key] = 1

print(mappings)

class WenJuanXing:
    def __init__(self, url):

        """
        self.wj_url = url
        self.post_url = None
        self.header = None
        self.cookie = None
        self.data = None

    def set_data(self):
        """
        这个函数中生成问卷的结果，可根据问卷结果，随机生成答案
        :return:
        """
        self.data = {
            'submitdata': '1$3}2$3}3$1}4$2}5$2}6$2}7$2}8$2}9$2}10$2'
        }

    def set_header(self):
        """
        随机生成ip，设置X-Forwarded-For
        ip需要控制ip段，不然生成的大部分是国外的
        直接利用随机数生成IP地址
        :return:
        """
        ip = '{}.{}.{}.{}'.format(112, random.randint(64, 68), random.randint(0, 255), random.randint(0, 255))
        self.header = {
            'X-Forwarded-For': ip,
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; WOW64; rv:68.0) Gecko/20100101 Firefox/68.0'
        }

    def get_ktimes(self):
        """
        随机生成一个ktimes,ktimes是构造post_url需要的参数，为一个整数
        :return:
        """
        return random.randint(15, 50)

    def get_response(self):
        """
        访问问卷网页，获取网页代码
        :return: get请求返回的response
        """
        response = requests.get(url=self.wj_url, headers=self.header)
        self.cookie = response.cookies
        return response

    def get_jqnonce(self, response):
        """
        通过正则表达式找出jqnonce,jqnonce是构造post_url需要的参数
        :param response: 访问问卷网页，返回的reaponse
        :return: 找到的jqnonce
        """
        jqnonce = re.search(r'.{8}-.{4}-.{4}-.{4}-.{12}', response.text)
        return jqnonce.group()

    def get_rn(self, response):
        """
        通过正则表达式找出rn,rn是构造post_url需要的参数
        :param response: 访问问卷网页，返回的reaponse
        :return: 找到的rn
        """
        rn = re.search(r'\d{9,10}\.\d{8}', response.text)
        return rn.group()

    def get_id(self, response):
        """
        通过正则表达式找出问卷id,问卷是构造post_url需要的参数
        :param response: 访问问卷网页，返回的reaponse
        :return: 找到的问卷id
        """
        id = re.search(r'\d{8}', response.text)
        return id.group()

    def get_jqsign(self, ktimes, jqnonce):
        """
        通过ktimes和jqnonce计算jqsign,jqsign是构造post_url需要的参数
        :param ktimes: ktimes
        :param jqnonce: jqnonce
        :return: 生成的jqsign
        """
        result = []
        b = ktimes % 10
        if b == 0:
            b = 1
        for char in list(jqnonce):
            f = ord(char) ^ b
            result.append(chr(f))
        return ''.join(result)

    def get_start_time(self, response):
        """
        通过正则表达式找出问卷starttime,问卷是构造post_url需要的参数
        :param response: 访问问卷网页，返回的reaponse
        :return: 找到的starttime
        """
        start_time = re.search(r'\d+?/\d+?/\d+?\s\d+?:\d{2}', response.text)
        return start_time.group()

    def set_post_url(self):
        """
        生成post_url
        :return:
        """
        self.set_header()  # 设置请求头，更换ip
        response = self.get_response()  # 访问问卷网页，获取response
        ktimes = self.get_ktimes()  # 获取ktimes
        jqnonce = self.get_jqnonce(response)  # 获取jqnonce
        rn = self.get_rn(response)  # 获取rn
        id = self.get_id(response)  # 获取问卷id
        jqsign = self.get_jqsign(ktimes, jqnonce)  # 生成jqsign
        start_time = self.get_start_time(response)  # 获取starttime
        time_stamp = '{}{}'.format(int(time.time()), random.randint(100, 200))  # 生成一个时间戳，最后三位为随机数
        url = 'https://www.wjx.cn/joinnew/processjq.ashx?submittype=1&curID={}&t={}&starttim' \
              'e={}&ktimes={}&rn={}&jqnonce={}&jqsign={}'.format(id, time_stamp, start_time, ktimes, rn, jqnonce, jqsign)
        self.post_url = url  # 设置url
        print(self.post_url)

    def post_data(self):
        """
        发送数据给服务器
        :return: 服务器返回的结果
        """
        self.set_data()
        response = requests.post(url=self.post_url, data=self.data, headers=self.header, cookies=self.cookie)
        return response

    def run(self):
        """
        填写一次问卷
        :return:
        """
        self.set_post_url()
        result = self.post_data()
        print(result.content.decode())

    def mul_run(self, n):
        """
        填写多次问卷
        :return:
        """
        for i in range(n):
            time.sleep(1)
            self.run()

Redis密码设置

发表于 2019-07-11

Redis的安装

在Ubuntu下安装redis很轻松

apt-get install update
apt-get install redis-server

之后输入命令

redis-server
redis-cli

即可确认是否安装成功

Redis密码的设置

进入redis命令行之后，输入 

config get requirepass

可以看到密码是空

config set requirepass 密码

即可设置密码

Redis的远程登录

仅设置密码还无法远程登录redis，因为redis一开始是只允许在本机登陆的，需要在redis的config里进行相应的设置。

  1. 注释掉Bind 127.0.0.1
  2. 将protected-mode 设为no
  3. 设置远程链接密码

错误信息

absolute paths not recommended in jsps.

这句话意思是在jsp文件中，不建议使用绝对路径来导入静态文件，如css，js等。

百度给出的结果

直接用$获得上下文路径

把

<script src="$/libs/jquery/jquery.min.js" ></script>

改为

<script src="${pageContext.request.contextPath}/libs/jquery/jquery.min.js" ></script>

即可

用JSP中的Java代码获取上下文路径

代码：

<%
String path = request.getContextPath();
String basePath = request.getScheme() + "://"
+ request.getServerName() + ":" + request.getServerPort()
+ path + "/";
%>

路径改为

<img src="<%=basePath%>images/head_success.jpg" height="80" width="650"/>

当web.xml中的webapp版本较低时，browser是无法解析 ${pageContext.request.contextPath} 的，浏览器指向的为$%7BpageContext.request.contextPath%7D，发生这种情况就是webapp版本过低，修改web.xml文件即可。

排序双指针法:

先讲一讲C++ sort函数的用法

对vector排序

sort(nums1.begin(),nums1.end());

对数组排序也是一样的用法，第一个参数为起始地址，第二个参数为最后一个元素的后一位的地址。

因为sort实际上是不去第二个参数所表明的地址的值的，就和vector.end()所指向的也不是最后一个值而是最后一个值的后一个位置一样。

删除C++中vector的重复元素

res.erase(unique(res.begin(), res.end()), res.end());

排序双指针法

class Solution {
public:
    vector<int> intersection(vector<int>& nums1, vector<int>& nums2) {
        sort(nums1.begin(),nums1.end());
        sort(nums2.begin(),nums2.end());
        vector<int> res;
        for(int i=0,j=0;i<nums1.size() && j<nums2.size();){
            if(nums1[i] == nums2[j]){
                res.push_back(nums1[i]);
                i++;
                j++;
            } else if(nums1[i]<nums2[j]){
                i++;
            }else
                j++;
        }
        res.erase(unique(res.begin(), res.end()), res.end());
        return res;
    }
};

C/C++中指针在函数中传递的问题

发表于 2019-05-24

正常的地址传递

众所周知，如果只通过普通的方式进行函数间传递的话是简单的值传递，无法在被调函数内修改参数的大小，但是如果用指针进行地址传递就可以修改，下面是一个简单的例子。

#include <iostream>
using namespace std;
void add(int *p);
int main(){
    int p;
    add(&p);
    cout << p;
    return 0;
}
void add(int *p){
    int in;
    cin >> in;
    *p = in;
}
输出结果为8.

指针的值传递

如果上方的程序改成这个样子，程序会没有输出（因为空指针）。

#include <iostream>
using namespace std;
void add(int *p);
int main(){
    int *p;
    add(p);
    cout << *p;
    return 0;
}
void add(int *p){
    int in;
    cin >> in;
    *p = in;
}

因为这个样子进行传递的话就相当于把指针进行了参数传递！！！
传递的不是指针的地址，而是指针的值，这就相当于是在函数之间进行值传递了，main函数里当然看不到add函数里给P指向的值。
